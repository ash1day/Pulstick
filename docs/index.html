Pulstick

A minimal metronome that lives in your menu bar. One click away, always ready.

Download for macOS View on GitHub

Requires macOS 15 (Sequoia) or later

Everything you need, nothing you don't

Menu Bar Resident

One-click access from your menu bar. No need to switch windows or hunt through your Dock.

BPM 40 -- 240

Wide tempo range covering everything from slow practice to fast exercises. Precise slider and step buttons.

Tap Tempo

Tap to set the tempo naturally. Match any song or rhythm by feel, not by numbers.

Accent Beats

Set time signatures with accented downbeats. Choose from 4/4, 3/4, and 6/8.

Dock-Free

Runs silently without a Dock icon. Your workspace stays clean and uncluttered.

Lightweight

Built natively for macOS with minimal resource usage. No Electron, no bloat.

Get started in seconds

1

Download

Grab the latest release from GitHub Releases.

2

Install

Move Pulstick.app to your Applications folder.

3

Launch

Open Pulstick. The metronome icon appears in your menu bar. Done.
